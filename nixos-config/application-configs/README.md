# Application configs

In this directory will be listed the configs for different applications.

## Files

- [default.nix](./default.nix): executed as the folder gets imported and loads all the other files contained inside the folder
- [wm_config.nix](./wm_config.nix): contains the configurations for both yabai and skhd (MacOS)
- [homebrew.nix](./homebrew.nix): contains all the homebrew configurations
